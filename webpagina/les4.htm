De vierde en laatste les van ons project elektronica en ICT bekeken we wat er allemaal mogelijk was met de Arduino die we de week ervoor zelf hadden gesoldeerd. We programmeerden kleine eenvoudige programmatjes. Zo zagen we wat er mogelijk was met de ledjes, de lichtsensor, de temperatuursensor en alle andere zaken op ons bordje.

We hebben zelf een stuk code gemaakt dat gebruik maakt van de sensor en een LED aan en uit zet. 


    void setup() {
        // put your setup code here, to run once:
      pinMode(A0,INPUT);
      pinMode(10,OUTPUT);
      Serial.begin(9600);
      
      }
      
      void loop() {
        // put your main code here, to run repeatedly:
      if(analogRead(A0) == 0){
        digitalWrite(10,HIGH);
      }else{
        digitalWrite(10,LOW);
      }
      
      Serial.println(analogRead(A0));
      
      }
